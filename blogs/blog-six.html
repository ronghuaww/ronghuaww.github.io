Hello everyone!

What are you most proud of doing last week?

I got project two done! Even though it was submitted a day late, it was a really happy moment when my partner and I finished debugging all the errors.

Definitely peer programming and communication. Going through each line of code one at a time, and reading the print statements to see if it matches with the intended output was so helpful. There were multiple instances where we caught bugs that the other programmer did not realize.

What impact did what you do have on other people?

I think working with my partner, we both got to learn a lot more about pointers and design implementations. It was so helpful to be able to talk it out with another person.

What did you think of Paper #6. Open-Closed Principle.

It was a fun read! Its really nice to read more extensively about the OOP concepts I have learned before. More specifically, it was really eye-opening to see how they used virtual classes for the open-closed principle.

What did you think of Project #2. Voting.

It was a fun project! Even though there was a lot of debugging, it was really fun practicing my skills using pointers and references. I also got the chance to practice using different data structures like vectors, lists, and arrays throughout this project which gave me more confidence in using these various data types.

What did you think of reverse(), iterator_tags, const methods and friend functions? (This question will vary, week to week.)

reverse() is such an easy concept, but implementation-wise, it was harder than I expected. Const is also a fun concept to learn, I definitely plan on using it more as it is so helpful to restrict changes made to a variable that you don’t want to change. It was interesting to learn about all the different types of iterators, I didn’t know there was so many types and how different data structures are connected with specific types. For friends functions, my friend had told me about their use before but I had a difficult time wrapping my head around it. After seeing it in class, it finally clicked!

What made you happy this week?

What's your pick-of-the-week or tip-of-the-week?
